📚 Documentation API - ENT

API REST pour l'École Nationale des Transmissions

Base URL: http://192.168.100.16:8000
CORS: Activé pour tous les domaines
Format: JSON

🎯 API Événements

GET /events/api/events/

Description: Récupère tous les événements publiés

Paramètres de requête (optionnels):
  • category - Filtrer par catégorie
  • featured - Événements en vedette (true/false)
  • upcoming - Événements à venir (true/false)

  "email": "john@ent.dz",
